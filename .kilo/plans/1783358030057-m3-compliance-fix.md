# Material 3 全量合规修正计划

## 目标
消除项目中所有 `Colors.grey`、弃用 API、非 M3 组件用法，统一使用 `colorScheme` + M3 组件。

## 验证
每完成一个任务后运行 `flutter analyze`，确保零 error。

---

## 任务列表

### Task 1: `app_theme.dart` — 补充全局主题配置
**文件**: `lib/theme/app_theme.dart`
**改动**:
- 添加 `dividerTheme` 和 `scrollbarTheme` 到两个 theme
- 添加 `inputDecorationTheme` 统一 M3 输入框圆角

```dart
dividerTheme: DividerThemeData(
  color: colorScheme.outlineVariant,
),
scrollbarTheme: ScrollbarThemeData(
  thumbColor: WidgetStateProperty.all(colorScheme.outlineVariant),
),
inputDecorationTheme: InputDecorationTheme(
  border: OutlineInputBorder(
    borderRadius: BorderRadius.circular(12),
  ),
),
```
注意：需要将 `lightTheme`/`darkTheme` 改为 getter 或工厂方法以访问 `colorScheme`，或直接使用 `ThemeData.raw` / 后处理。最简方案：把主题声明改为函数 `lightThemeOf()` 内部先构造 `colorScheme` 再用它构造 `ThemeData`。
