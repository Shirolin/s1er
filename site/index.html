功能

为长时间读帖而做

对接 Discuz! Mobile API，Material Design 3 界面，把论坛常用路径收进一套客户端。

浏览与阅读

版块、主题列表、分页与楼层定位；常用 BBCode、引用、代码与帖内图片，深色模式下也能长时间刷帖。

  • 分页跳转
  • 楼层定位
  • BBCode
  • 图片查看

发帖与互动

回复、引用、新主题与编辑；投票、评分与举报，流程贴合社区习惯。

麻将脸表情

原版 png/gif 入库打包；面板 local-first，缺失再走 CDN。

消息与关系

私信、系统提醒、好友列表、收藏与每日签到，常用互动不跳出客户端。

主题与本地体验

亮暗色与多套种子色、字号调节；阅读历史、回复草稿、本地黑名单与 L1 备份，数据留在你的设备上。

  • 种子色
  • 阅读历史
  • L1 备份
  • 黑名单

界面

深色优先，桌面与手机同构

桌面端分栏读帖，手机端沉浸刷楼；同一套 Material 3 色阶与组件语言。

平台

一套代码，验收逐步推进

仓库含 Web、Android、iOS、Windows、macOS 与 Linux 工程。当前已验证 Web 调试、Android APK 与 Windows zip；其余平台可自构建，尚未做系统验收。发布包见 GitHub Releases 或夸克网盘。

  • Web 已验证 · 调试
  • Android 已验证 · APK
  • Windows 已验证 · zip
  • iOS 未验证
  • macOS 未验证

独立第三方，非官方产品

本项目与 Stage1st 官方无隶属、授权或背书关系。使用时须遵守 Stage1st 的服务条款与社区规则；论坛接口或权限变化可能影响部分功能。

Cookie 在本地加密存储；密码不落盘。详见 隐私政策。
