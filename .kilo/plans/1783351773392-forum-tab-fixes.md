# Forum Tab: 修复统计数字 + 分类折叠功能

## 问题分析

### Bug: 帖子/回复统计始终为 0

根因在 `lib/services/api_service.dart:112-118`。`parseForumList` 构建分类树时，从 `catlist` 创建的顶层分类硬编码了 `threads: 0, posts: 0`：

```dart
categories.add(ForumCategory(
  fid: catFid,
  name: catName,
  description: '',
  threads: 0,    // ← 永远是 0
  posts: 0,      // ← 永远是 0
  subforums: subforums,
));
```

Discuz! 的 `catlist` 本身不带帖子统计，统计信息在 `forumlist` 的各子版块中。需要从 `subforums` 聚合。

### 新功能: 分类折叠 + 记忆

需要：
1. `_ForumCategoryTile` 改为 StatefulWidget，点击头部切换展开/折叠
2. 折叠状态持久化到 Hive `settings` box（项目已有此 box）
3. 使用 AnimatedSwitcher 或 AnimatedCrossFade 做展开/折叠动画

## 实施计划

### Task 1: 修复统计聚合

**文件**: `lib/services/api_service.dart` — `parseForumList` 方法

在构建 categories 时，遍历 subforums 聚合 threads/posts：

```dart
int totalThreads = 0;
int totalPosts = 0;
for (final sub in subforums) {
  totalThreads += sub.threads;
  totalPosts += sub.posts;
}

categories.add(ForumCategory(
  fid: catFid,
  name: catName,
  description: '',
  threads: totalThreads,
  posts: totalPosts,
  subforums: subforums,
));
```

### Task 2: 折叠状态持久化 Provider

**文件**: `lib/providers/settings_provider.dart`

在 `AppSettings` 中新增 `Set<String> collapsedForums` 字段，`SettingsNotifier` 中读写 Hive：

```dart
// AppSettings 新增
final Set<String> collapsedForums;

// SettingsNotifier 新增方法
void toggleForumCollapse(String fid) {
  final collapsed = Set<String>.from(state.collapsedForums);
  collapsed.contains(fid) ? collapsed.remove(fid) : collapsed.add(fid);
  state = state.copyWith(collapsedForums: collapsed);
  Hive.box('settings').put('collapsedForums', collapsed.toList());
}
```

### Task 3: 分类头部可点击折叠

**文件**: `lib/screens/home_screen.dart` — `_ForumCategoryTile`

- 从 `ConsumerWidget` 改为 ConsumerWidget（读 settings provider 获取折叠状态）
- 头部 Container 包裹 InkWell，点击触发 toggleForumCollapse
- 折叠时隐藏子版块列表，展开时显示
- 头部右侧添加展开/折叠箭头图标（`Icons.expand_more` / `Icons.chevron_right`，AnimatedRotation 做旋转动画）
- 使用 `AnimatedSize` 或 `AnimatedCrossFade` 做子版块列表的展开/折叠动画
